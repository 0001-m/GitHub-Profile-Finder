# GitHub Profile Finder

A React app that lets you search any GitHub user and view their profile info.

## Features
- Search any GitHub username
- Displays avatar, bio, followers, and public repos
- Loading and error states
- Clean dark UI with Tailwind CSS

## Tech Used
- React
- Tailwind CSS
- GitHub Public API

## What I learned
- How to fetch data from a real API using async/await
- Managing multiple state variables in React
- Handling loading and error states
- Deploying a React app with Vercel
